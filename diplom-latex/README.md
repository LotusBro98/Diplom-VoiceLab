## Installing Linux ##
1) INSTALL TEX LIVE
```
sudo apt-get install texlive-full
```
2) INSTALL TEXMAKER
```
sudo apt-get install texmaker
```
3) INSTALL CYRFONTS
```
sudo apt-get install scalable-cyrfonts-tex
```
## Usage ##
```
sh build.sh
```
